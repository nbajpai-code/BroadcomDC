# Broadcom Data Center Storage Resources

Broadcom is a leader in storage connectivity, providing the industry-standard MegaRAID controllers and Host Bus Adapters (HBAs).

## 1. RAID Controllers (MegaRAID)

### MegaRAID 9600 Series (Gen 4.0 / 24G SAS)
*   **Tri-Mode SerDes:** Supports NVMe, SAS, and SATA devices simultaneously.
*   **Key Models:**
    *   **MegaRAID 9670-24i:** 24 internal ports, x8 host interface.
    *   **MegaRAID 9660-16i:** 16 internal ports.
*   **Features:** Hardware Secure Boot, RAID 0, 1, 5, 6, 10, 50, 60.
*   **Documentation:** [MegaRAID 9600 Series User Guide](https://docs.broadcom.com/doc/96xx-MR-HBA-Tri-Mode-UG)

### MegaRAID 9500 Series (Gen 4.0 / 12G SAS)
*   **Key Models:**
    *   **MegaRAID 9560-16i:** 8GB Cache, PCIe Gen 4.0.
    *   **MegaRAID 9580-8i8e:** Hybrid internal/external ports.
*   **Documentation:** [MegaRAID 9500 Series User Guide](https://docs.broadcom.com/doc/95xx-MR-HBA-Tri-Mode-UG)

## 2. Host Bus Adapters (HBAs)

### HBA 9600 Series
*   **Purpose:** High-performance connectivity for software-defined storage (vSAN, Ceph) and external JBODs.
*   **Key Models:**
    *   **eHBA 9600-24i:** Enhanced HBA with RAID features.
    *   **HBA 9600-16e:** External connectivity.
*   **Documentation:** [HBA 9600 Series Data Sheet](https://docs.broadcom.com/doc/12354774)

### HBA 9500 Series
*   **Purpose:** Standard enterprise connectivity.
*   **Key Models:**
    *   **HBA 9500-16i:** 16-port internal.
    *   **HBA 9500-8e:** 8-port external.
*   **Documentation:** [HBA 9500 Series Data Sheet](https://docs.broadcom.com/doc/12354773)

## 3. NVMe/PCIe Switches

### PEX8800 Series (Gen 4.0 PCIe Switch)
*   **Features:** Low latency, high port count for NVMe-oF and AI/ML clusters.

## 4. Management Software

*   **LSI Storage Authority (LSA):** Web-based management software for monitoring and configuring RAID/HBA.
*   **StorCLI:** Command line interface for scripting and automation.
