# Broadcom Data Center Security Resources (Symantec)

Broadcom's Symantec division provides critical security for physical and virtual data center infrastructure.

## Symantec Data Center Security: Server Advanced (DCS:SA)
*   **Purpose:** Hardens and protects physical and virtual servers.
*   **Key Features:**
    *   **Host-Based Intrusion Detection/Prevention (HIDS/HIPS):** Monitoring and blocking of malicious activity.
    *   **Application Control:** Whitelisting/blacklisting of applications.
    *   **File Integrity Monitoring (FIM):** Detects unauthorized changes to critical files.
    *   **Configuration Monitoring:** Ensures compliance with security policies.
*   **Documentation:** [DCS:SA 6.9.3 Administration Guide](https://techdocs.broadcom.com/us/en/symantec-security-software/endpoint-security-and-management/data-center-security-server-advanced/6-9-3.html)

## Critical System Protection (CSP)
*   **Purpose:** Agent-based security for legacy and embedded OS environments (e.g., ATM, POS, Industrial Control Systems).
*   **Key Features:**
    *   **System Locking:** Prevents unauthorized software execution.
    *   **Memory Protection:** Protects against buffer overflows and memory injection attacks.

## Symantec Endpoint Protection (SEP) for Data Center
*   **Purpose:** Antivirus and endpoint protection optimized for high-density virtual environments.
*   **Key Features:**
    *   **Virtual Image Exception:** Optimizes scanning performance in VDI/VSI environments.
    *   **Network Threat Protection:** Inspects incoming and outgoing traffic.
