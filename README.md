# Broadcom Data Center Resources

This repository contains comprehensive resources, documentation links, and product details for Broadcom's Data Center portfolio.

    *   **Fibre Channel:** Brocade Gen 7 (X7 Directors, G730/G720) and Gen 6 (G620) Switches.
*   **[Storage](STORAGE.md)**

    *   **RAID Controllers:** MegaRAID 9600 & 9500 Series.

*   **[VMware Cloud Foundation](VMWARE.md)**

## Overview

Broadcom's Data Center solutions span the entire infrastructure stack, from the physical layer (PHYs, Optical) to the chip level (Switching ASICs), adapter level (NICs, HBAs), and up to the software layer (VMware Cloud Foundation, Symantec Security).
